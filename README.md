# Readme

This repo contains a devcontainer setup and is intended to be used as a template.

Notes:
- Open the `devcontainer` directory in VS Code and use "Reopen in Container".
- The container mounts the repo root at `/home/dev/dev_container`.

Quick start:
- Clone the template repo.
- Open the `devcontainer` directory in VS Code.
- Choose "Reopen in Container" when prompted.

CLI usage:
- `cd devcontainer`
- `make up` to build and start the container.
- `make shell` to enter the container.
